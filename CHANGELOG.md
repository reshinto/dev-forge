# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-09

### Added

- 10 generalized review agents (code reviewer, tech lead, QA, UI/UX, technical writer, silent failure hunter, code simplifier, code explorer, system architect, product strategist)
- 13 workflow skills (TDD, debugging, feature development, implementation planning, verification, accessibility audit, architecture review, documentation review, security/coverage audit, strict TypeScript review, README optimization, branch safety check, system management)
- 10 plugin hooks with self-gating (branch safety, AI attribution blocking, branch naming enforcement, TypeScript checks, accessibility checks, Python checks, Java checks, hardcoded wait banning, auto-PR after push)
- Project scaffold system with adaptive `init.sh` and diff-based `update.sh`
- Comprehensive documentation (README + 10 docs/ files)
